# Mark Quest as Done

1.  Navigate to the **Quest Board** in `BattleView`.
2.  Locate your assigned quest in the **In Progress** column.
3.  Click the **Mark as Done** button.
4.  A form will appear asking for **Proof of Work**.
5.  Enter a brief comment describing what you did (e.g., "Fixed the login bug", "Added the new API endpoint").
6.  Click **Submit**.
7.  The quest status will change to **Pending Review**.
8.  Wait for the **Game Master (Admin)** to review and authorize the completion.

> **Note:** Only the Admin can finally complete the quest to award XP and damage the Boss.
